# SSYCloud Tap

## How do I install these formulae?

`brew install ssycloud/tap/<formula>`

Or `brew tap ssycloud/tap` and then `brew install <formula>`.

For this project:

```bash
brew install ssycloud/tap/assemble-flow
```

`assemble-flow` installs the CLI only. If you also want the Codex or Claude skill pack, use the GitHub Release installer from `AssembleFlow`.
